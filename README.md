# Dungeons & Dragons Hobby Website

## Description
A multi-page website about the tabletop role-playing game Dungeons & Dragons.
The site was created as a web development project and presents some small fictional stories about original characters in a fantasy setting world.

The project focuses on fundamental front-end web development concepts
including page structure, navigation, and visual styling.

## Features
- Multiple interconnected pages
- Semantic HTML structure
- Custom styling using CSS
- Navigation between sections of the site
- Organized layout and content presentation

## Technologies
- HTML
- CSS

## Purpose
This project was created as part of coursework in the
Computer Programming and Analysis program at Fanshawe College.
